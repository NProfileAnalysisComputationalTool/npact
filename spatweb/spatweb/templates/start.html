NPACT - N-Profile Analysis Computational Tool

N-PACT is now in advanced stage of development. File upload and "Start Analysis!" will produce a graphical output showing sequence compositional contrasts (as in frame analysis), regions of non-random compositional contrasts, and gene predictions, plus the correspoding text files. The "Configure Analysis" page is currently unstable as its functionality is progressively being developed.

Run NPACT for frame-analysis and gene prediction from a sequence file in the Genbank format (see a sample record or the About page).

The NPACT server can process uploaded files, files fetched from a url, or genbank data pasted directly in. Only one of the following is neccessary
