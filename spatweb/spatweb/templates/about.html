N-PACT About

You reached a version in advanced development of N-PACT, a program designed to identify protein-coding regions by frame analysis (Bibb et al. 1984) in nucleotide sequences of any length. N-PACT also performs quantitative frame analysis on the sequence, identifying sequence regions of significant compositional contrasts and corresponding gene predictions (Oden and Brocchieri 2012).

The N-PACT server produces a (multi-page) PostScript file representing frame-specific GC (or any nucleotide) content for frame analysis (Ishikawa and Hotta 1999; Brocchieri et al. 2005), published gene annotation when available, high-scoring segments and new gene predictions.

Getting Started

To start the analysis upload a nucleotide sequence file in the GenBank format using any of methods provided. Directly start the analysis with default parameters clicking "Get Analysis!" or change options with "Configure Analysis" before launching it. The program accepts as input a GenBank-formatted sequence text file that minimally must include the filed DEFINITION and one line starting with the word ORIGIN, which idenifies the beginning of the nucleotide sequence. All alphabetic characters following the ORIGIN line are interpreted as sequence characters. Any alphabetic character other than A/a, C/c, G/g, T/t, U/u is interpreted as unknown, and all non-alphabetic characters are ignored. All annotated coding sequences are read from the file up to the line ORIGIN from " CDS " lines, following a GenBank format. Gene names are assigned reading them from the variables "gene" (first choice) or "locus_tag" (second choice).

N-profile and prediction PostScript output

The multi-page PostScript ouput file produced by N-PACT contains the following information:

Newly Annotated. These are coding regions predicted by the N-PACT procedure, represented by arrows pointing in the 5'-to-3' direction. Newly predicted regions are in bold-face whereas regions corresponding to published genes but with modified starting position are displayed in lighter color. The names of the coding regions are in the form: X-number[-/*]Y, where:

T identifies the type of hit and assumes values:

H: H-type hit, identified using a scoring scheme based on compositional expectations of coding regions.

G: G-type hit, identified by non-random association of base type with codon position.

number: A unique identification number followed by:

'-', if significance does not reach global threshold (less strong contrast).

'*', if ignificance reaches global threshold (stronger contrast).

Y is an upper or lower case character indicating the type of predicted start-of-translation codon. Upper case indicates that the predicted codon is within 15 positions of the start of the hit, lower case indicates that it is more distant. The potential start-of-translation codons looked for are, in the order:

ATG (A/a) or GTG (G/g)

TTG (T/t)

CTG (C/c)

ATT (W/w)

Annotation. These are coding regions reported in the annotation of the GenBank input file, from the CDS field (see above).

Hits. Graphical representation of high scoring segments identified as colored segments on two tracks, corresponding to direct strand (upper track) and complementary strand (lower track).

Profiles. These curves represent by default GC content within a moving window of 201 nucleotides. Usage of other nucleotide type combinations can be chosen by the user. Each curve represents percent frequencies within a subsequence composed of every third nucleotide of the genome, starting, respectively, from position 1 (red), 2 (green) or 3 (blue). A light-gray curve indicates the overall composition of the genome within corresponding intervals.

Coloring scheme

The coloring scheme used throughout the output follows the following rules. The nucleotide sub-sequences starting, respectively, from position 1, 2 or 3 of the sequence - composed of positions (i mod 3) where i = 1, 2 or 3 - are assigned by defalut color red (R), green (G), and blue (B), respectively. Annotated genes, newly predicted genes, and high-scoring segments, are assigned the same color of the sub-sequence on which their third codon positions are located.
