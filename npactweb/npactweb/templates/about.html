N-PACT About

About

N-PACT is a computational tool designed to identify protein-coding regions in a nucleotide sequence of any length and composition, using a new quantitative approach extending the principle of visual frame analysis (Bibb et al. 1984). N-PACT identifies sequence segments of statistically significant compositional contrast, and uses them for gene prediction (Oden and Brocchieri 2012, in preparation).

The N-PACT server produces a multi-page graphical output representing, along the sequence, frame-specific GC (or any nucleotide) content for frame analysis (Ishikawa and Hotta 1999; Brocchieri et al. 2005), published gene annotations (when available), high-scoring segments and new gene predictions.

Getting Started

To start the analysis upload a nucleotide sequence file in the Genbank format, using any of the methods provided. Directly start the analysis with default parameters clicking "Get Analysis!", or change options with "Configure Analysis". The program accepts as input a Genbank-like sequence text file that, minimally, must include a header with the fields DEFINITION, describing the sequence, and ORIGIN, with following lines containing sequence data. Starting from the line subsequent to ORIGIN all alphabetic characters are interpreted as sequence characters. Any alphabetic character other than A/a, C/c, G/g, T/t, U/u is treated as unknown, and all non-alphabetic characters are ignored. Optional annotated coding sequences are read from the file from the " CDS " fields, following the Genbank format. Gene names are assigned reading them from the variables "gene" (first choice) or "locus_tag" (second choice).

N-profile and prediction graphical output

N-PACT produces a multi-page graphical ouput in Postscript or PDF format. This output contains the following information:

Newly Annotated. These are coding regions newly predicted by the N-PACT procedure, represented by arrows pointing in the 5'-to-3' orientation. Newly predicted coding regions are in bold-face, whereas coding regions corresponding to published genes but with modified starting position are displayed in lighter color. The names of the coding regions are in the form: [H/G]-number[-/*][A/a/G/g/T/t/C/c/W/w], for example, H-2335*A, G-123-t, where:

[H/G] identifies the type of hit and assumes values:

H: H-type hit, identified using a scoring scheme based on compositional expectations of coding regions.

G: G-type hit, identified by non-random association of base type with codon position.

number: A unique identification number followed by:

'-', if significance does not reach global threshold (less strong contrast).

'*', if ignificance reaches global threshold (stronger contrast).

[A/a/G/g/T/t/C/c/W/w] is an upper or lower case character describing the predicted start-of-translation codon. Upper case indicates that the predicted codon is within 15 codons of the start of the hit, lower case indicates that it is more distant. The potential start-of-translation codons looked for are, in the order:

ATG (A/a) or GTG (G/g)

TTG (T/t)

CTG (C/c)

ATT (W/w)

Input file CDS. Coding regions reported in the annotation of the Genbank input file, from the CDS field (see above).

Hits. Graphical representation of high-scoring segments identified as colored segments on two tracks, corresponding to direct strand (upper track) and complementary strand (lower track).

Profiles. These curves represent by default GC content within a moving window of 201 nucleotides. The user can choose to display other nucleotide type combinations (e.g., purines). Each curve represents percent frequencies within a subsequence composed of every third nucleotide of the input sequence, starting, respectively, from position 1 (red), 2 (green) or 3 (blue). A light-gray curve indicates the overall composition of the genome within corresponding intervals.

Coloring scheme

The coloring scheme used throughout the graphical output follows the following rules. The nucleotide sub-sequences starting, respectively, from position 1, 2 or 3 of the sequence - composed of positions (i mod 3) where i = 1, 2 or 3 - are assigned by defalut color red (R), green (G), and blue (B), respectively. Annotated genes, newly predicted genes, and high-scoring segments, are assigned the same color of the sub-sequence corresponding to their third codon positions.
