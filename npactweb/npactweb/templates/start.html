NPACT - N-Profile Analysis Computational Tool

N-PACT identifies, in a nucleotide sequence of any length, regions of significant period-three (frame-specific) compositional contrast, and uses them to predict coding sequences. N-PACT can be used for identifying compositional contrasts, for de novo gene prediction, or for supporting or revising sequence annotations obtained with other methods. The N-PACT combined quantitative and visual frame-analysis is most effective in sequences of high GC content (e.g., > 55%), but can also be used for sequences of any composition. In contrast to visual frame analysis, N-PACT uses a four-nucleotide alphabet, determines the statistical significance and precise boundaries of regions of contrast, and takes into consideration the ORF structure and local compositional properties of the sequence.

To use N-PACT, input a sequence file in the Genbank format (see the About page) using any of the methods provided, and "Start Analysis!" to obtain output text files and a graphical representation showing frame-specific sequence composition, genes annotated in the original file (when available), sequence regions with compositional contrast (hits), and new gene predictions. Parameters and plot properties can be modified with "Configure Analysis".
