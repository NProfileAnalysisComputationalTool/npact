NPACT - N-Profile Analysis Computational Tool (Prokaryotes)

For quantitative frame-analysis and identification of ORFs with statistically-significant 3-base periodicities in prokaryotic sequences NPACT is currently accepting sequence files in the GenBank format (see a sample record or the About page).

After seleting a sequence file, "Start Analysis!" will result in a graphical output showing sequence compositional contrasts for frame analysis, gene annotation (when available), and, by default, regions of non-random compositional contrast (hits), newly-identified ORFs, and corresponding output text files. Parameters can be modified using "Configure Analysis", leading to a page where the user can choose which nucleotide type(s) to represent for frame analysis, start and end position of the analysis, and level of significance of high scoring segments (hits). From this page the user can also disable identification of hits and corresponding ORFs, in which case N-PACT will only produce the graph of frame-specific nucleotide content and positions of genes annotated in the input file when available.

The NPACT server can process uploaded files, files fetched from a url, or genbank data pasted directly in. Only one of the following is neccessary.
